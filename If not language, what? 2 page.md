## The Stack That Keeps Failing

```
  1950s  Intelligence is computation.     → Input → [rules]       → Output
         Symbolic AI. Expert systems.       Collapsed 1980s. Brittle.

  1980s  Intelligence is pattern match.   → Input → [weights]     → Output
         Neural networks. Deep learning.   Hallucinations. No world model.

  2017+  Intelligence is language scale.  → Tokenₙ → [transformer] → Tokenₙ₊₁
         LLMs. The current paradigm.       Failing now. $1bn per critique.

  THE UNTOUCHED BASE:    What IS intelligence?
                         Not what produces it. What it fundamentally IS.
                         Nobody answered this.
```

LeCun attacks the top. He is right to. The base is still untouched. Until it is replaced, the next architecture hits the same wall in a different location.
